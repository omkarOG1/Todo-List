# My Todo App

A simple Todo List built with HTML, CSS, and JavaScript.

## Features
- Add new todos
- Delete todos
- Stores todos in localStorage
- Fully responsive UI

## How to Use
1. Type your todo in the input box
2. Click on “Add”
3. Click “Remove” to delete a todo

## Technologies Used
- HTML
- CSS
- JavaScript
